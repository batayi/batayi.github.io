Topographic Change Classification from Multi-Temporal Point Clouds

Unsupervised 3D change detection · Siamese KPConv · DeepCluster

3D Point Clouds
MSc Thesis – University of Bonn & TUM · Study area: Isar River, Germany
Data: UAV photogrammetry & TLS point clouds – two epochs

Overview

This MSc thesis focuses on classifying topographic changes along a river corridor using multi-temporal high-resolution point clouds. I used an Encoder-Fusion Siamese KPConv backbone with a DeepCluster-style unsupervised learning pipeline to distinguish erosion, deposition, vegetation change, and stable areas.

Data & Preprocessing

  • Two epochs of 3D point clouds from UAV photogrammetry and Terrestrial Laser Scanning.
  • M3C2-based change metrics used as reference for binary change/no-change masks.
  • Cylindrical sampling of the scene into local 3D neighborhoods for training.
  • Hand-crafted geometric features (planarity, omnivariance, height rank, etc.).

Method

  • Encoder-Fusion Siamese KPConv network for learning 3D embeddings of before/after point clouds.
  • DeepCluster-style loop to cluster embeddings and refine pseudo-labels.
  • Post-processing and mapping of clusters to geomorphic classes (erosion, deposition, vegetation change, stable).

Results

The model successfully highlighted subtle erosion and deposition patterns beyond binary change maps, revealing spatially coherent geomorphic units. It demonstrates how unsupervised 3D deep learning can support river monitoring and topographic change analysis without dense manual labels.
